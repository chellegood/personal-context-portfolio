## Daily Tools

### Claude (Pro)
Primary AI assistant and increasingly my daily operating system. Use for 
thinking, writing, building, and working through problems. Actively building 
toward using Claude Cowork and Claude Code more regularly. This context 
portfolio is being built here and will live in a Claude Project.

### Wispr Flow
Voice-to-text tool used regularly on my MacBook Air. Core part of how I 
interact with my laptop.

### Google Calendar
Primary calendar. Have a personal calendar plus a shared calendar with Alex. 
This is where my schedule lives. Note: Apple Calendar also used on iPhone — some events end up there. 
Google Calendar is the source of truth but Apple Calendar is worth 
checking for completeness.

### Apple Reminders
Current to-do list and task management tool. Have personal lists plus shared 
lists with Alex (including grocery list). I can add to it quickly from iPhone 
or MacBook. Staying with it for now due to convenience and Alex integration, 
but skeptical about how well it will connect to Claude and Minkus long-term. 
Open to switching if a better solution emerges.

### Lovable
Vibe coding tool — used to build the Leucadian Loaf website, bachelorette 
sites, and personal app projects. Starting point for most of my build projects.

### OpenClaw
The technology powering Minkus, my personal AI executive assistant. Currently 
being onboarded.

### GitHub
Code and file storage. Currently using for the personal context portfolio repo 
(forked from nlwhittemore/personal-context-portfolio).

### Vercel
Used to deploy two projects: Sam's bachelorette site and Doni's bachelorette 
site. Account active.

### Perplexity (Pro)
AI-powered search. Pro account active.

### Apple Notes
De facto note-taking app due to convenience on iPhone. Not ideal — I know I 
need a better system for creating and tracking documents. Also using plain 
TextEdit files on laptop increasingly, especially for AI-related work.

### Browser & Search
Primary browser: Chrome. Primary search engine: Google for straightforward 
searches. Tends to use Gemini or ChatGPT for search-like queries to preserve 
Claude credits for deeper work.

### MacBook Air (16GB)
Primary computer.
